# F1 Telemetry Dashboard (Static + Optional Backend)

This repository contains a production-ready static frontend for an F1 telemetry dashboard and optional instructions to run a self-hosted OpenF1 backend via Docker Compose.

- Frontend: static ES modules, deployable to GitHub Pages.
- Optional backend: OpenF1 community server + MongoDB (Docker Compose) for WebSocket push and stable CORS.

See the two Word documents provided with this package:
- F1_Telemetry_Dashboard_GitHub_Procedure.docx
- F1_Telemetry_Dashboard_Full_Project_Backend.docx
