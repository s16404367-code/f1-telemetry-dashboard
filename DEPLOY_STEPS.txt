🏁 F1 TELEMETRY DASHBOARD - DEPLOYMENT IN 3 SIMPLE STEPS
========================================================

STEP 1: CREATE GITHUB REPOSITORY (2 minutes)
============================================

1. Go to: https://github.com/new
2. Fill in the form:
   - Repository name: f1-telemetry-dashboard
   - Description: Professional Formula 1 Telemetry Dashboard
   - Make it PUBLIC (important!)
3. Click "Create repository"
4. You'll see a page with instructions
5. COPY the URL (looks like: https://github.com/YOUR-USERNAME/f1-telemetry-dashboard.git)


STEP 2: UPLOAD FILES TO GITHUB (2 minutes)
==========================================

Option A: Using GitHub Web Interface (Easiest)
----------------------------------------------
1. Go to your new repository
2. Click "Add file" → "Upload files"
3. Drag and drop the entire f1-dashboard folder contents
4. Scroll down and click "Commit changes"
5. Done!

Option B: Using Git Command Line (If you have Git installed)
------------------------------------------------------------
1. Open terminal/command prompt
2. Navigate to f1-dashboard folder
3. Run these commands:

   git init
   git add .
   git commit -m "Initial commit: F1 Telemetry Dashboard"
   git branch -M main
   git remote add origin https://github.com/YOUR-USERNAME/f1-telemetry-dashboard.git
   git push -u origin main

4. Done!


STEP 3: ENABLE GITHUB PAGES (1 minute)
======================================

1. Go to your repository: https://github.com/YOUR-USERNAME/f1-telemetry-dashboard
2. Click "Settings" (top right)
3. Click "Pages" (left sidebar)
4. Under "Source":
   - Select: "main" branch
   - Select: "/" (root) folder
5. Click "Save"
6. Wait 1-2 minutes for deployment


YOUR DASHBOARD IS NOW LIVE!
===========================

Access it at:
https://YOUR-USERNAME.github.io/f1-telemetry-dashboard/

Replace YOUR-USERNAME with your actual GitHub username.


WHAT TO DO NEXT
===============

1. Open the URL in your browser
2. Click "Load 2026 Season"
3. Select a meeting (e.g., Bahrain)
4. Select a session (e.g., Race)
5. You should see:
   - Track map with driver dots
   - Timing tower with standings
   - Driver information panel

✅ Success! Your F1 Telemetry Dashboard is live!


TROUBLESHOOTING
===============

If you see "404 Page Not Found":
- Wait 2-3 minutes (GitHub Pages can be slow)
- Clear browser cache (Ctrl+Shift+Delete)
- Refresh the page (F5)
- Check that your repository is PUBLIC

If nothing loads when you select a session:
- Check your internet connection
- Try a different session
- Open browser console (F12) and look for errors

If the page looks broken:
- Try a different browser
- Clear browser cache
- Refresh the page
